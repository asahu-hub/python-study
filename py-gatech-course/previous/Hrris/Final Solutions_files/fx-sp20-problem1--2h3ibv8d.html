Background

Suppose you have a long report or document to read but limited time. Which sentences are the most important? In this notebook, you'll formulate the task of finding key sentences as a linear algebra problem, and see if such an automatic method can help.

Idea. Let's start with the following intuitive idea:

A sentence is important if it contains many important words. And a word is important if it appears in many important sentences.

At first glance, this idea seems a bit circular! But it turns out we can formulate it as a well-defined mathematical problem in the language of linear algebra.

Data representation. Suppose the document consists of nn distinct sentences and mm unique words. Next, let ii be a word and let jj be a sentence. Furthermore, let nini denote the number of sentences containing word ii. We can represent the document by a matrix AA, called the term-sentence matrix, where each entry ai,jai,j is defined as

ai,j=1ln(n+1ni)0if word i appears in sentence j, orotherwise.ai,j={1ln(n+1ni)if word i appears in sentence j, or0otherwise.

Here, lnxlnx denotes the natural logarithm of xx. Observe that ai,jai,j tends to be small if ii does not appear in many sentences (i.e., nini is small). The n+1n+1 ensures that we do not divide by zero when calculating ai,jai,j; in particular, even if word ii appears in every sentence (ni=nni=n), the value lnn+1ni>0lnn+1ni>0. Lastly, observe that since each sentence presumably contains just a few of the possible words. As such, we expect AA to be sparse.

A mathematical formulation of the task. Given word ii and sentence jj, let's denote word ii's importance score by wiwi; and similarly, let sjsj denote the importance score of sentence jj.

We said we think these scores are inter-related; let's posit the following linear model of these relationships:

wisjjai,jsjiaj,iwi,wijai,jsjsjiaj,iwi,

where the symbol means proportional to. So, the first formula says that if wiwi appears in sjsj, meaning ai,j0ai,j0, then wiwi's score increases in proportion to sjsj's score. The second formula is interpreted analogously.

If we know wiwi and sjsj for every word ii and every sentence jj, then the most important words and sentences should have the largest scores. Thus, similar to the webpage ranking problem of Notebook 11, we can use these scores to rank words or sentences.

You should recognize right away that these two formulae can be written in matrix form. Letting w=[w0,w1,,wm1]w=[w0,w1,,wm1] be the (column) vector of word scores and s=[s0,s1,,sn1]s=[s0,s1,,sn1] be the (column) vector of sentence scores, we can define ww and ss as

cwwcss==AsATw,cww=Ascss=ATw,

where cwcw and cscs are two unknown constants.

Going one step further, let's plug these two equations into one another to obtain the following:

(AAT)w(ATA)s==(cscw)w(cwcs)s.(AAT)w=(cscw)w(ATA)s=(cwcs)s.

You should immediately recognize an eigenvalue problem! In particular, cscwcscw is an eigenvalue of the mmmm matrix AATAAT, whose corresponding eigenvector is ww. Similarly, cwcs=cscwcwcs=cscw is also an eigenvalue of the nnnn matrix ATAATA, whose corresponding eigenvector is ss.

Suppose we were to give you a singular value of AA---say, 00, the largest one---and the corresponding left and right singular vectors---say, u0u0 and v0v0, respectively. You should be able to figure out how to obtain ww and ss from u0u0 and/or v0v0. (Notebook 15, anyone?)

Aside: Recall that the singular values are always non-negative. Furthermore, since the entries of AA are also non-negative, if 00 is the largest singular value, then we can assume u0u0 and v0v0 are also non-negative. You'll need this fact later.

An algorithm. From the background above, we now have a "simple" algorithm to rank words and sentences.

  1. Given a document, construct the word-sentence matrix AA.
  2. Compute the largest (or principal) singular value and corresponding singular vectors of AA. Denote the singular value by 00 and the left- and right-singular vectors as u0u0 and v0v0, respectively.
  3. Use u0u0 and v0v0 to rank words and sentences. (You need to figure out the relationship between ww and ss and u0u0 and v0v0!)

def clean(s):
    from re import finditer
    pattern = r"[a-z]+('[a-z])?[a-z]*"
    return [match.group(0) for match in finditer(pattern, s.lower())]

# Your task: Complete this function!
def gen_coords(bags, word_to_id):
    # Some code to help you get started:
    m, n = len(word_to_id), len(bags)
    rows, cols, vals = [], [], []
    
    # You should build rows, cols, and vals:
    ### BEGIN SOLUTION
    from numpy import zeros
    from math import log
    
    # Compute the `n_i` values
    num_docs = zeros(m)
    for b in bags:
        for w in b:
            num_docs[word_to_id[w]] += 1
            
    # Construct a coordinate representation
    for j, b in enumerate(bags): # loop over each sentence j
        for w in b: # loop over each word i in sentence j
            i = word_to_id[w]
            a_ij = 1.0 / log((n+1) / num_docs[i])
            rows.append(i)
            cols.append(j)
            vals.append(a_ij)
    ### END SOLUTION
    
    # Returns your arrays:
    return rows, cols, vals

def ex2__avl82n38vOI(bags, word_to_id):
    from numpy import zeros
    from math import log
    m, n = len(word_to_id), len(bags)
    rows, cols, vals = [], [], []
    num_docs = zeros(m)
    for b in bags:
        for w in b:
            num_docs[word_to_id[w]] += 1
    for j, b in enumerate(bags):
        for w in b:
            i = word_to_id[w]
            a_ij = 1.0 / log((n+1) / num_docs[i])
            rows.append(i)
            cols.append(j)
            vals.append(a_ij)
    return rows, cols, vals

def get_svds_largest(A):
    from scipy.sparse.linalg import svds
    from numpy import abs
    u, s, v = svds(A, k=1, which='LM', return_singular_vectors=True)
    return s, abs(u.reshape(A.shape[0])), abs(v.reshape(A.shape[1]))
